# OntoPhraseology-MWEs

An OntoLex-Lemon-based Linked Open Data resource for Chinese–French phraseological units and multiword expressions.

## Overview

OntoPhraseology-MWEs is an open research repository dedicated to the representation of Chinese–French phraseological units using Semantic Web technologies and the OntoLex-Lemon model.

The repository contains:

- a multilingual phraseological ontology;
- linguistic datasets;
- RDF resources;
- corpus data;
- notebooks used for data processing and ontology construction;
- documentation describing the data model.

The repository is intended for research in:

- Computational Phraseology
- Lexicography
- Linguistic Linked Open Data (LLOD)
- Knowledge Graphs
- Multilingual NLP
- Large Language Models

---

## Research context

This repository was initiated during a Short-Term Scientific Mission (STSM) hosted by the **CIRCSE Research Centre**, Università Cattolica del Sacro Cuore (Milan, Italy), in June 2026.

The research stay was funded by **COST Action CA21167 – UniDive (Universality, Diversity and Idiosyncrasy in Language Technology)**.

The repository continues to be maintained and extended by CHEN Lian.
